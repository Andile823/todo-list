This is a simple Todo List App built with HTML, CSS, and JavaScript.
Users can add, delete, and manage tasks easily.

## Features
- Add new tasks ✅
- Delete completed tasks ❌
- Simple and clean UI 🎨

## How to run it
Just open the `index.html` file in your browser.
